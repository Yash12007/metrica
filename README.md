# Metrica
**Metrica** is a lightweight Windows performance analysis tool designed to visualise system metrics and simplify basic system operations with a clean, modern interface.
Built for speed, clarity, and control.

---

## Features
- Real-time performance monitoring  
  - CPU usage  
  - Memory (RAM) usage  
-  Lightweight & fast  
  - Minimal background footprint  
-  Simple system handling  
  - Focused on essential insights  
-  Modern UI  
  - Clean visuals, distraction-free  
-  Offline-first  
  - No forced internet dependency
